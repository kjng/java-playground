Work Item configuration data is used to configure basic aspects of work items for this project. The following can be configured:

Types and Attributes: Define work item types and custom attributes.

Enumerations: Define enumerations for work item attributes.

Attribute Customization: Define default values, value sets, calculated values and validation rules.

Workflows: Define workflows for work items.

Editor Presentations: Define editor layouts for work items.

Quick Information Presentations: Configure presentations for the work item editor's Quick Information section.

Predefined Queries: Specify work item queries available to all users of the project area.

Query Editor Presentations: Associate work item attributes with presentations for query parameters.

Approval Tracking: Associate approval state changes to workflow actions to be triggered.

Templates: Specify work item templates available to all users of the project area.

Change Management Type Binding: Define the mapping of conceptual change management types like 'Defect' and 'Plan Item' to actual work item types.

E-mail Templates: Customize E-mails that are sent to users.
